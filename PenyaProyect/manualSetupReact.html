Manual: Integrar React en Laravel 11

Este manual detalla los pasos necesarios para integrar React en un proyecto Laravel 11 ya creado y generar un primer componente React de prueba.

1. Requisitos previos

  • Un proyecto Laravel 11 ya creado.
  • Docker configurado con un servidor Nginx y PHP.
  • NPM instalado en tu máquina.

2. Instalación de dependencias

Ejecuta los siguientes comandos en la raíz de tu proyecto:

npm install react react-dom
npm install --save-dev @vitejs/plugin-react

3. Configuración de Vite para React

Edita el archivo vite.config.js en la raíz del proyecto. Si no existe, créalo y añade lo siguiente:

import { defineConfig } from 'vite';
import laravel from 'laravel-vite-plugin';
import react from '@vitejs/plugin-react';

export default defineConfig({
    plugins: [
        laravel({
            input: ['resources/js/app.jsx'],
            refresh: true,
        }),
        react(),
    ],
});

4. Crear el punto de entrada de React

En la carpeta resources/js, crea un archivo llamado app.jsx con el siguiente contenido:

import React from 'react';
import ReactDOM from 'react-dom/client';

function App() {
    return 
Hello, World!

}

const root = ReactDOM.createRoot(document.getElementById('app'));
root.render();

5. Configurar la vista de Laravel

Edita o crea un archivo de vista Blade, por ejemplo, resources/views/VIEW_SELECTED.blade.php, y añade lo siguiente:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Laravel React</title>
    @vite('resources/js/app.jsx')
</head>
<body>
    <div id="app"></div>
</body>
</html>

6. Compilar los recursos

Ejecuta el siguiente comando para compilar los recursos del frontend:

npm run dev

Si deseas generar una compilación para producción, utiliza:

npm run build

7. Verificar la integración

Accede a la URL del proyecto (http://localhost:8080/ROUTE_SELECTED. Deberías ver el mensaje Hello, World! en la pantalla.
