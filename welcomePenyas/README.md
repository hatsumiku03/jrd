## About
Rembember to run a:
    
    npm install

To get all the depencies :3

Copy .env.example to .env

    cp .env.example .env

And generate a encrypted key:
    
    php artisan key:generate

Maybe you need to do a

    composer install

Or
    
    composer update

Finally, remember all is in docker containers, you must execute

    docker compose up -d

To build up all
